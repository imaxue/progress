##### if else的简写方法

##### 如果代码块里面只有一条语句就可以简写  省略大括号
// 如果if... else 语句大括号省略  只会找到与他最邻近的第一条语句

```
/如果代码块里面只有一条语句就可以简写 省略大括号
    // 如果if ... else 语句大括号省略,  只会找到与他最邻近的第一条语句
    var score = 91 // 写法一
    if (score > 90) {
      alert('优秀')
    } else {
      alert ('良好')
    }

    // 写法二
    if (score > 90) alert('优秀'); else alert('良好');
```
