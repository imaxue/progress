- 方式一
  - ubuntu
    + 编辑文件
        ```bash
        $ vim /etc/iptables/rules.v4
        ```
    + 加入这行，8080指需要开放的端口
        ```vim
         -A INPUT -p tcp -m state --state NEW -m tcp --dport 8080 -j ACCEPT
        ```
    + 立即生效命令
        ```bash
        $ iptables-restore < /etc/iptables/rules.v4
        ```
